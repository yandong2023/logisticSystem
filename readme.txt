这是一个物流调试系统，可以实现中心系统对每个手持客户端的快递人员的统一调度、分析、入库等操作。具体来说：
一、手机端
    1、快递人员接到公司发的快件，不需要自己去操作，只需要打开手机app，按照app替你规划好的最短路线去送快递即可
送完点确定，会向服务器端提交完成的指令。
   2、手机app在地图上显示你当前的位置，已送多少家，还剩多少家，并根据你的送货速度，估算出你多长时间能够送完，包括提成多少，工资多少等。
二、服务器端
   1、系统会对所有的订单进行分析处理，可以划分地区，根据快递人员进行分工，并算出最佳送货路线
   2、时时显示快递人员的送货情况
